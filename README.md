# Lab-Automation
A full-stack laboratory workflow automation platform designed to streamline operational processes, record management, and user task handling.
The system reduces manual workload by automating database operations, structuring workflows, and enabling role-based access across multiple portals.

🔹 Key Features
Role-based authentication system (Admin, Staff, Customer portals)
Automated CRUD operations for structured data handling
Workflow tracking and task management modules
File upload and document storage functionality
Modular architecture with shared components
Composer-based dependency management

🔹 Tech Stack
PHP
MySQL
JavaScript
Composer

🔹 Purpose
Built to demonstrate full-stack backend architecture, authentication flows, and database-driven workflow automation in a real-world operational scenario.
